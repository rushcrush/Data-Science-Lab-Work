📊 Student Performance Analysis
Description
Analysis of student data to understand factors affecting final grades (G3).

Dataset
Student dataset with study habits, demographics, and grades (G1, G2, G3).

Key Insights
Study time positively impacts performance
G2 strongly predicts final grade (G3)
Most students score in the average range
